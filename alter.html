ALTER TABLE

This command is used to Add/Change/Modify/Drop existing structure of the table.
  • ADD Column
  • Enable/Desable Constraints
  • Change Column
  • Modify Column
  • Drop Column
ADD Column- When a new column is to be added to the table structure without constraints.
Syntax:-
ALTER TABLE table_name
ADD COLUMN column_name data type;
Add more than one column
Syntax:-
ALTER TABLE table_name
ADD COLUMN column_name data type;
ADD COLUMN column_name data type;
ADD Column by Position
  • Last (by default)
  • First
  • After
Syntax:-
ALTER TABLE table_name
ADD COLUMN column_name data type (size) FIRST;
Syntax:-
ALTER TABLE table_name
ADD COLUMN column_name data type (size) AFTER column_name;
ADD Column - When a new column is to be added to the table structure with constraints. Syntax:-
ALTER TABLE table_name
ADD COLUMN column_name data_type (size) Constraint_name, ADD constraint_name column_name;
When integrity constraints have to be included. Syntax:-
ALTER TABLE table_name
ADD CONSTRAINT constraint_name column_name;
Change Column- This is used to change name and data type of an existing column. Without Constraints:-
Syntax:-
ALTER TABLE table_name
CHANGE COLUMN old_column_name new_column_name new_data_type (size);
Change more than one column- Syntax:-
ALTER TABLE table_name
CHANGE COLUMN old_column_name new_column_name new_data_type (size);
CHANGE COLUMN old_column_name new_column_name new_data_type (size);
Change column With Constraints:- Syntax:-
ALTER TABLE table_name
CHANGE COLUMN old_column_name new_column_name new_data_type (size) Constraint_name
ADD constraint_name (column_name); MODIFY COLUMN- This is used to modify size of the data type or the data type itself of an existing column without shanging column name.
Syntax:-
ALTER TABLE table_name
MODIFY COLUMN column_name data_type (size);
DROP COLUMN - When column in a table need to delete
Syntax:-
ALTER TABLE table_name
DROP COLUMN column_name ;
When removing constraint from a column. Syntax:-
ALTER TABLE table_name
DROP constraint_name column_name;
DROP TABLE
This command is used to delete/remove table from the database. Syntax:-
DROP TABLE table_name;
TRUNCATE TABLE
When we only want to delete the data inside the talbe, and not the table itself.
Syntax:-
TRUNCATE TABLE table_name;
RENAME TABLE
This command is used to rename one or more table.
Syntax:-
RENAME TABLE old_table_name to new_table_name;

ALTER DATABASE

ALTER DATABASE enables you to change the overall characteristics of a database. These characteristics are stored in the db.opt file in the database derectory. To use ALTER DATABASE, you need the ALTER privilege on the databse.
ALTER SCHEMA is a synonym for ALTER DATABASE.
The database name can be omitted from the syntax, in which case the statement applies to the default database.
Syntax:-ALTER DATABASE daftabase_name;

DROP DATABASE

The DROP DATABASE statement is used to delete a database.
Syntax:-DROP DATABASE daftabase_name;

UPDATE

The UPDATE statement is used to update existing records in a table.
Syntax:-
UPDATE table_name
SET column1=value1,column1=value2...
WHERE some_column = some_value;
Note-WHERE is necessary otherwise all records will be replace with given value.

UPDATE with CASE

UPDATE table_name
SET new_column =
CASE
WHEN column_name1 = some_value1 THEN new value1
WHEN column_name2 = some_value2 THEN new value2
ELSE new_value3
END;
ON UPDATE
  • ON UPDATE CASCADE
  • ON UPDATE CASCADE
  • ON UPDATE CASCADE
  • ON UPDATE CASCADE
ON UPDATE is use after foreign key when creating a table for update the table after creating table

DELETE

The DELETE statement is used to delete records in a table.
1. Delete a Specific Record Syntax:-
DELETE FROM table_name
WHERE some_column = some_value;

Note-WHERE is necessary otherwise all records will be delete.

Unable to Delete Parent Table = This used to delete the content of a Table.
1st a fall we have to delete child table contents of a row
DELETE FROM child_table
WHERE condition;
DELETE FROM child_table
WHERE condition;
After that we can delete the content of a parent by using this syntax:-
DELETE FROM parent_table
WHERE condition;
ON DELETE
  • ON DELETE CASCADE
  • ON DELETE SET NULL
  • ON DELETE NO ACTION
  • ON DELETE RESTRICTION
All ON DELETE functions are used after foreign key when creating table for deleteing, restriction from deleting etc. after creating tables.

COPY old Table to new Table

Within Same Database
Syntax:-
CREATE TABLE new_table LIKE old_table;
INSERT new_table SELECT * FROM old_table;

Within different Database
Use the database where you want to copy the old table
Syntax:-
CREATE TABLE new_table LIKE old_db.old_table;
INSERT new_table SELECT * FROM old_db.old_table;

CROSS JOIN/Cartesian Join/Cartesian Product/Cross Product

The Cross Join returns every row from one table crossed with every row from the second.
Syntax:-
  • SELECT * FROM Employee CROSS JOIN Department;
  • SELECT Name,depname FROM Employee CROSS JOIN Department;
  • SELECT * FROM Employee, Department;
