=RDBMS avoided the navigation model as in old DBMS and introduced introduced Relational model. Teh relational model has relationship between tables using primary keys, foreign keys and indexes. Thus the fetching and storin of data become faster than zthe old Navigational model. RDBMS is useful to efficiently manage vast amoun of data and is used inlarge business application. Example:-SQL Server, Oracle, MySQL, MariaDB, SQLite.

Structured Query Language (SQL)

=It is commonly used with all relational database for data definition and manipulation.
    Feature of SQL:-
  • It is a non procedural language.
  • It is an English-like language.
  • It can process a ingle record as well as set of records at a time.
  • All SQL statment define what is to be done rather than how it is to be done.
  • SQL has facilities for defining database views, security, transaction etc.
