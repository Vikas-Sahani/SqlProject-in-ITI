Where

WHERE is used to search for specific data.

Conditions of Where

1) Specific data from all column
Syntax:-
SELECT * FROM table_name
WHERE column_name operator 'value';

2) Specific data from specific column
Syntax:-
SELECT column_name FROM table_name
WHERE column_name operator 'value';

Note:-Value can be text or numeric. If it is text then we have to put single quotes.

NULL VALUE

NULL values represent missing unknown data. NULL= not equal to zero

IS NULL-This is used to select only the records with NULL values in the column.
Syntax:-
SELECT column_name FROM table_name
WHERE column_name IS NULL;


IS NOT NULL -This is used to select only the records with no NULL values in the column.
Syntax:-
SELECT column_name FROM table_name
WHERE column_name IS NULL;

OR

The OR operator displays a record if either the first condition OR the second condition are true.
Syntax:-
SELECT * FROM table_name
WHERE column_name = 'value'
OR column = 'value';

AND OR

Syntax:-
SELECT * FROM table_name
WHERE column_name = 'value'
AND(column_name1 = 'value' OR column_name1 = 'value');

IN

The IN operator allows you to specify multiple values in a WHERE clause. Syntax:-
SELECT * FROM table_name
WHERE column_name IN ('value1','value2',...);

NOT IN

Syntax:-
SELECT * FROM table_name
WHERE column_name NOT IN ('value1','value2',...);

BETWEEN

The BETWEEN operator selects values within a range. The values can be numbers, text or dates. 1. Between Number Syntax:-
SELECT * FROM table_name
WHERE column_name BETWEEN value1 AND value2;

2. Between Text Syntax:-
SELECT * FROM table_name
WHERE column_name BETWEEN "value1" AND "value2;" 3. Between Date Syntax:-
SELECT * FROM table_name
WHERE column_name BETWEEN 'yyyy/mm/dd' AND 'yyyy/mm/dd';

NOT BETWEEN

To display the data which is not in the range.

1.NOT BETWEEN Number Syntax:-
SELECT * FROM table_name
WHERE column_name NOT BETWEEN value1 AND value2;
2. Between Text Syntax:-
SELECT * FROM table_name
WHERE column_name NOT BETWEEN "value1" AND "value2;"
3. Between Date Syntax:-
SELECT * FROM table_name
WHERE column_name NOT BETWEEN 'yyyy/mm/dd' AND 'yyyy/mm/dd';

BETWEEN with IN

Syntax:-
SELECT * FROM table_name
WHERE (column_name BETWEEN value1 AND value2) AND column_name IN (value1,value);

BETWEEN with NOT IN

Syntax:-
SELECT * FROM table_name
WHERE (column_name BETWEEN value1 AND value2) AND NOT column_name IN (value1,value);

LIKE

The LIKe operator is used to search for aspecified pattern in a column. Syntax:-
SELECT * FROM table_name
WHERE column_name LIKE 'pattern';

Wildcards

Wildcards are used to search for data within a table. These characters are used with the LIKE operator.
  1. %:-Zero or more characters

    'Geek%' -All starting with geek Ex:-Geekyshows
    '%shows'-All ending with shows. Ex:-Geekyshows
    '%sh%'-All containing with sh. Ex:-Geekyshows
  2. _:-One single charachter

    'Show_':-Starting with show then any character. Ex:-shows
    '_eek':-any character then eek. Ex:- geek
    'G_e_k':-G then any character, then e then any character, then k Ex:-Geek

NOT LIKE

Syntax:-
SELECT * FROM table_name
WHERE column_name NOT LIKE 'pattern';

ORDER BY

This is used to sort the records.
ASC-It sorts in ascending order (by default)
DESC- It sorts in descending order.
  1. Sort in descending order
    Syntax:-
    SELECT * FROM table_name
    ORDER BY column_name DESC;
  2. Sort in ascending order
    Syntax:-
    SELECT * FROM table_name
    ORDER BY column_name ASC;(ASC is optional)
