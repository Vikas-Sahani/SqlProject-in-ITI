Select

The SELECT statement is used to select data from adatabase and retrieve the information.
  • Select all columns from the table
    • Syntax:-SELECT * FROM table_name;
  • Select Particular columns from the table
    • Syntax:-SELECT column_name1, column_name2,...
    FROM table_name;
SELECT LIMIT is used to specify some numbr of records to disply.
Syntax:-SELECT column_name1,column_name1
FROM table_name
LIMIT records_number;
Explanation=here is records_number is not display and other records are displaying.

Select Distinct

The SELECT DISTINCT statement is used to display only distinct (different) values.
Syntax:-SELECT DISTINCT column_name FROM table_name;
