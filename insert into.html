Intsert Into

The INSERT INTO statement is used new records/row/tuple ina table.

Syntax:-INSERT INTO table_name (column1, column2, column3, column4, ...)
VALUES (value1, value2, value3, value4 ...);


Rules:-
  • Column and Value order should be sae.
  • Any value that goes into a VARCHAR, CHAR, DATE or TEXT column has single quotes around it. There are no need of quotes for numeric values (INT, DEC)

Conditions of Insert Into

1)Without specifying column name
Syntax:-INSERT INTO table_name
VALUES (value1, value2, value3, value4,...);

2)Changing the order of column
Syntax:-INSERT INTO table_name (column2, column1, column3, column4, ...)
VALUES (value2, value1, value3, value4,...);

3)Insert Data only in Specified Columns
Syntax:-INSERT INTO table_name(column1, column2, column3)
VALUES (value1, value2, value3);

4)Insert multiple record at one time.
Syntax:-INSERT INTO table_name(column1, column2, column3, column4)
VALUES(value1, value2, value3, value4),
(value1, value2, value3, value4),
(value1, value2, value3, value4);

Single Quotes Problem

PROBLEM=INSERT INTO my_tab(t_id, c_name, address
VALUES (142, 'K.K's Company', 'Delhi');
There are tow way to solve this problem:-
  • Use backslash
    • Ex:-'K.K\'s company'
  • Use two time single quotes
    • Ex:-'K.K''s Company'
