SHOW DATABASES/ TABLES

  • SHOW DATABASES
    • :-This command is used to view all the databases name.
    Syntax:-SHOW DATABASES;


  • SHOW TABLES
    • :-This command is used to view all the tables of current database.
    Syntax:-SHOW DATABASES;



USE

 This is use to tell your RDBMS/DBMS that you want to use this database.
Syntax:-USE database_name;

SHOW COLUMNS

It shows all the columns of table and their data type along with any other column specific details. It is just like DESC table_name.
Syntax:-
SHOW COLUMNS FROM table_name;

SHOW CREATE DATABASE

It shows commands which you have written while creating your Database.
Syntax:- SHOW CREATE DATABASE database_name;

SHOW CREATE TABLE

Syntax:-
SHOW CREATE TABLE table_name;
